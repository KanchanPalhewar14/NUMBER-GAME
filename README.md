# Number Game 🎯
This is a simple Java-based Number Guessing Game developed as part of the CodSoft Internship.

# Features
- Random number generation
- User input using Scanner
- Hints (Too high / Too low)
- Limited attempts

#Technologies Used
- Java

#How to Run
1. Open terminal
2. Compile:
   javac NumberGame.java
3. Run:
   java NumberGame
